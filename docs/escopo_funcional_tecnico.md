# Escopo Funcional e Técnico Inicial

## Funcionalidades MVP
- Cadastro por faixa etária
- Atribuição automática da fase do Trivium
- Jogos Radix e Spelling Bee
- Sistema de conquistas (troféus, medalhas)
- Painel de pais

## Técnicas
- Backend: Python (Django ou FastAPI)
- Frontend: Streamlit, React ou HTML básico
- Banco de dados: PostgreSQL ou SQLite
- API para carregar listas de palavras por categoria
- Gamificação: sistema de pontos, ranking e progresso

## Estrutura Modular
Cada jogo e cada fase deve ser independente e expansível.
