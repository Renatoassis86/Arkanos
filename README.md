# ARKANOS – O Reino do Saber e da Virtude

Plataforma online de jogos educacionais para crianças e adolescentes baseados na Educação Cristã Clássica. Enraizada nos princípios da Verdade, Bondade e Beleza, e estruturada nas fases do Trivium: Gramática, Lógica e Retórica.

## Linguagens e Tecnologias
- 🐍 Python (Django/FastAPI)
- 🖥️ HTML/CSS/JS
- 🔠 React ou Streamlit (frontend alternativo)
- 📦 PostgreSQL ou SQLite (banco)
- 🎮 Pygame / custom JS games para minigames

## Organização
- `docs/`: documentos de planejamento e arquitetura
- `backend/`: APIs, modelos e lógica de negócio
- `frontend/`: páginas e interações visuais
- `data/`: listas de palavras e vocabulário de jogos
